== Description ==

This plugin allows you to add social share buttons to your posts. The plugin is flexible and easy to use.

<h3>Key Features</h3>
* Supports Facebook, Twitter, LinkedIn, Email, Pinterest share buttons.
* Typography setting
* Color setting
