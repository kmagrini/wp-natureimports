== Description ==

This plugin allows you to showcase your blog posts in a beautiful slider with multiple options, It is responsive ready so it will work perfectly on different devices like mobile and iPad. There is an option in the plugin from which you can show the posts on the basis of category. You can sort the order of posts in ascending or descending. There are useful options to enable and disable the post meta. The important feature of the plugin is providing the typography and color settings from where you can change the typography and color of the content.

<h3>Key Features</h3>

* Typography setting
* Color setting
* Navigation speed
* Posts order
* Number of posts
* Posts per page
* Enable/Disable Post meta
